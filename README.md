# Number Guessing Game
A simple command-line number guessing game built with Python.
## Features
* Random number generation
* User-friendly hints
* Limited guessing attempts
* Play again option
* Input validation
## Technologies Used
* Python 3
* Random Module
* Time Module
## How to Run
1. Open the project folder.
2. Run:
python Guess#.py
3. Follow the on-screen instructions.
## Learning Outcomes
* Conditional Statements
* Loops
* Functions
* Random Number Generation
* User Input Handling
